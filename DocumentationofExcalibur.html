Documentation of Excalibur

Excalibur is a language made in 2025 by Noah Shipley of the Caldera Organization. It is a beginner-friendly programming language made with Python and creativity.

Excalibur is made to be beginner-family, as their commands to add and subtract are simply:

add: 1 + 2

subtract: 1 - 2

That is enough, though, so let’s get started.

Beginning to Learn

If Excalibur is your first language, then Python might be a great second language to learn as Excalibur is derived off of Python (made with it).

But, if we want to learn Excalibur, we should run some commands.

Open up your interpreter and type:

write: “Hello World”
